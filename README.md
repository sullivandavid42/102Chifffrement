102Chifffrement
===============

Chiffrement par clef matricielle.

Pour chiffrer un message : 
     ./102chiffrement "mes_to_crypt", key (4 int), "BASE", 0

Pour déchiffrer : 
     ./102chiffrement "MES_TO_DECRYPT", key (4 int), "base", 1
